# Gestor de Fazenda

Projeto completo com suporte offline-first, Supabase, React + Vite, Dexie, React Query e PWA.

## Instalação
```
npm install
npm run dev
```

## Configuração
Criar arquivo `.env`:
```
VITE_SUPABASE_URL=xxxx
VITE_SUPABASE_ANON_KEY=xxxx
```

## Deploy para GitHub
Após baixar:
```
git init
git add .
git commit -m "feat: projeto inicial"
git branch -M main
git remote add origin https://github.com/ronaldofns/gestor-fazenda.git
git push -u origin main
```
